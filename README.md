# Simple Login Page

A clean, responsive login interface built with HTML, CSS, and a touch of JavaScript.

## 🚀 Features
* **Custom Styling:** Styled with CSS for a modern "card" look.
* **Responsive Design:** Optimized for different screen sizes.
* **Validation:** Basic input handling via `script.js`.

## 🛠️ Technologies Used
* **HTML5:** Structure and content.
* **CSS3:** Layout and styling (including box-shadows and border-radius).
* **JavaScript:** Functional logic.

## 📁 Project Structure
* `index.html`: The main structure.
* `style.css`: All the visual styling you see in the code.
* `script.js`: Interactive elements.

## 📝 How to Run
1. Clone this repository.
2. Open `index.html` in any web browser.
